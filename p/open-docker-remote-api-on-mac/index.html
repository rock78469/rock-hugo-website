Open Docker Remote API on Mac

解決在MacOS上的Docker Desktop 無法使用Docker Remote API問題

Mar 17, 2023
閱讀時間: 1 分鐘

說明

因為在MacOS上安裝的Docker Desktop 無法使用Docker Remote API

因此可以使用 socate image 去開啟這項服務

1
2
3

docker run -d --name socat --restart always \
 -p 2375:2375 \
 -v /var/run/docker.sock:/var/run/docker.sock alpine/socat tcp-listen:2375,fork,reuseaddr unix-connect:/var/run/docker.sock

修改環境變數

1
2
3

$vim ~/.bash_profile
export DOCKER_HOST=tcp://localhost:2375
$source .bash_profile

確認運行狀況

1

curl localhost:2375/version
proxy cdn
